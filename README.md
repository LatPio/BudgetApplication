# BudgetApplication
To run this applcation you need create .env (/budget/.env) file with parameters listed bellow: 

//Djanog stuff
  SECRET_KEY= /n
  DEBUG=
//Database access
  ENGINE=
  NAME=
  USER=
  PASSWORD=
  HOST=
//Your e-mail account for Sending e-mail
  EMAIL_HOST=
  EMAIL_HOST_USER=
  EMAIL_HOST_PASSWORD=

In requirements.txt are listed used packages for this project.
